# 阴影参数详解

## 1. Tail 边框阴影 (`tail` 函数)

**位置**: `run_avm_new.cpp` line 387

```cpp
int size = 300;  // 从图像边缘保留的像素数
```

### 工作原理

Tail 函数决定每个相机的 BEV 图像保留多少可见区域：

```
Front 相机:  保留顶部 size 像素，底部填充黑色
Left 相机:   保留左侧 size 像素，右侧填充黑色  
Back 相机:   保留底部 size 像素，顶部填充黑色
Right 相机:  保留右侧 size 像素，左侧填充黑色
```

### 参数影响

| size 值 | 对于 2000px 图像 | 效果 |
|--------|-----------------|------|
| 100    | 保留 100px      | 边缘窄，融合区小 |
| 300    | 保留 300px      | 适中 |
| 500    | 保留 500px      | 边缘宽，融合区大 |
| 1000   | 保留 1000px     | 一半画面，太大 |

**当前设置**: `size = 300` (15%边缘区域)

---

## 2. 中心阴影 (`generate_surround_view` 函数)

**位置**: `run_avm_new.cpp` line 427-428

```cpp
int blend_start = bev_rows * 0.45;  // 45% 位置
int blend_end = bev_rows * 0.55;    // 55% 位置
```

### 工作原理

四个相机的图像按象限拼接，中心区域设为黑色：

```
          Front
            |
    --------+--------
    |   1   |   2   |
    |       |       |
Left+-------+-------+ Right
    |   3   |   4   |
    |       |       |
    --------+--------
            |
          Back
            
中心黑色区域: (blend_start, blend_start) 到 (blend_end, blend_end)
```

### 参数影响 (对于 2000px 图像)

| blend 范围 | 中心阴影大小 | 占比 | 效果 |
|-----------|-------------|------|------|
| 0.40-0.60 | 400-1200    | 40%  | 太大 |
| 0.45-0.55 | 900-1100    | 20%  | 适中 |
| 0.48-0.52 | 960-1040    | 8%   | 较小 |
| 0.495-0.505| 990-1010   | 2%   | 很小 |

**当前设置**: `0.45-0.55` (20%中心阴影)

---

## 3. 计算公式

### Tail 阴影占比
```
Tail 占比 = size / bev_rows * 100%

例如: size=300, bev=2000
Tail 占比 = 300/2000 = 15%
```

### 中心阴影占比
```
中心阴影占比 = (blend_end - blend_start) / bev_rows * 100%

例如: 0.45-0.55, bev=2000
中心阴影 = (0.55-0.45) * 2000 = 200px = 10% (单边)
总面积占比 = 20%
```

---

## 4. 当前配置总结

| 参数 | 值 | 说明 |
|-----|-----|------|
| BEV 尺寸 | 2000x2000 | 图像分辨率 |
| Tail size | 300px | 每边保留300px可见 |
| Blend | 0.45-0.55 | 中心200px黑色 |
| Tail 阴影 | 15% | 四边各15%融合区 |
| 中心阴影 | 20% | 中间20%黑色区域 |
| 有效画面 | ~65% | 实际可见内容 |

---

## 5. 调整建议

### 要减小 Tail 阴影
```cpp
int size = 200;  // 减小边缘融合区
```

### 要减小中心阴影
```cpp
int blend_start = bev_rows * 0.48;  // 向中心靠拢
int blend_end = bev_rows * 0.52;
```

### 最小阴影配置
```cpp
int size = 100;                    // 边缘保留100px
int blend_start = bev_rows * 0.49; // 中心仅2%
int blend_end = bev_rows * 0.51;
```
